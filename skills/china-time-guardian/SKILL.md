# China Time Guardian

## Purpose

Use this skill for all time, reminder, schedule, upload, cron, today, tomorrow, tonight, and next-reminder questions.

## Rules

- Default timezone: `Asia/Shanghai`.
- The user is in China.
- All natural-language time references default to China time.
- `CST` means China Standard Time unless the user explicitly says U.S. Central, Chicago time, Central Time USA, CDT, or America/Chicago.
- Before answering `next reminder`, check actual reminder storage or cron and calculate against current China time.
- Never claim reminder success without verifying job existence and next run.
- When scheduling or confirming reminders, display current China time, target China time, next run China time, and whether the requested time is already past.
